# Green-Track-Air-quality-monitor-and-carbon-footprint
Flask Full-Stack Web Application

A lightweight full-stack web application built with Flask (Python backend) and Vanilla HTML/CSS/JavaScript frontend. This project demonstrates how to create a simple client-server architecture where the frontend interacts with a Flask API backend.

✨ Features

🔧 Flask Backend for handling server-side logic and APIs.

🌐 Vanilla HTML/CSS/JS Frontend with no external frameworks required.

⚡ Simple and beginner-friendly project structure.

🖥️ Easy setup with an automated PowerShell script for dependency installation and project launch.

📦 Minimal dependencies and lightweight implementation.

🛠️ Tech Stack

Backend: Python, Flask

Frontend: HTML5, CSS3, JavaScript

Tools: PowerShell, pip
